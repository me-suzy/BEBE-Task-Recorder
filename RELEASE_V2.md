### Installation

1. Download `Bebe - Task Recorder - Version 2.0.exe` from the Assets section below
2. Run the executable (it will automatically request administrator privileges)
3. Start recording!

### Usage

1. **Start Recording**: Click "Start Recording" button
2. **Perform Actions**: Move mouse, click, type keyboard input
3. **Stop Recording**: Press `ESC` or `F9` from any application
4. **Playback**: Click "Play" button to replay your recorded task
5. **Pause**: Press `F10` from any application during playback
6. **Continuous Playback**: Check "Run continuously until ESC/F9" for looping

### Keyboard Shortcuts

- **ESC / F9**: Stop recording or stop playback
- **F10**: Pause/Resume playback (works globally from any application)
- **Space**: Play/Resume playback
